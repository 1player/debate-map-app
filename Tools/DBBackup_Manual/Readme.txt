Before running "CreateBackup_JSON[...].ps1", you need to:
1) Install the "node-firestore-import-export" tool: npm i -g node-firestore-import-export.
2) Create and download an admin-sdk key (json): https://github.com/jloosli/node-firestore-import-export#retrieving-google-cloud-account-credentials
3) Store the json key at the path found within the ps1 scripts.
